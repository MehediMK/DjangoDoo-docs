DjangoDoo - A Modular Django Framework üöÄ

Welcome to DjangoDoo, an open-source, modular and extensible multipurpose application framework, inspired by Odoo and built using Django.

DjangoDoo is designed to provide a flexible and scalable architecture for building business applications, ERP systems, and other enterprise-grade solutions.

üåü Key Features

‚úÖ Modular System - Install and enable modules dynamically without modifying core code.
‚úÖ Flexible UI Views - Supports List View, Kanban View, Graph View, and Form View.
‚úÖ Real-time Updates - OnChange and Compute methods similar to Odoo for real-time calculations.
‚úÖ Extensible Architecture - Easily add new modules with models, views, templates, and APIs.
‚úÖ Multi-Tenant Support (Upcoming) - Enable different workspaces for different users.
‚úÖ User Role Management - Fine-grained access control for different user groups.
‚úÖ Django ORM & PostgreSQL - Powerful backend with optimized query performance.
‚úÖ API Ready - RESTful API support using Django REST Framework (DRF).

üéØ Why DjangoDoo?

DjangoDoo aims to bring the power and flexibility of Odoo to Django-based projects.
Instead of building monolithic applications, DjangoDoo allows developers to create modular applications where each feature is a separate module.

This approach provides:
üîπ Scalability ‚Äì Handle large enterprise applications.
üîπ Maintainability ‚Äì Keep modules independent and reusable.
üîπ Customizability ‚Äì Easily modify and extend features without breaking the core.

üöÄ Getting Started

Want to set up DjangoDoo? Follow our Getting Started Guide. 

git clone https://github.com/MehediMK/djangodoo.git
cd djangodoo
python -m venv venv
source venv/bin/activate  # On Windows, use venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

For more details, check the Installation Guide.
